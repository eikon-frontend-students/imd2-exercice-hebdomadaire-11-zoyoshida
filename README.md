# Exercice hebdomadaire 11

## Installation

1. Cloner le repository
2. Ouvrir le dossier avec Visual Studio Code
3. Ouvrir une fenêtre de terminal dans Visual Studio Code et entrer la commande suivante :

```bash
npm install
```

4. Installer fontawesome. Pour cela, aller sur la [fontawesome.com](https://fontawesome.com), créer un kit et l'ajouter dans le _head_ de index.html.
   ☝️ Vous pouvez réutiliser le kit de la semaine dernière.

## Travail

Depuis le terminal, entrer la commande suivante :

```bash
npm run dev
```

## Consigne

1. Ouvrir [la maquette Figma](https://www.figma.com/design/Id63zVpguVBOMexfAYPWGR/imd2-exercice-hebdomadaire-11?node-id=0-1&m=dev) dans l'équipe de votre année
2. Intégrer la maquette aussi précisément que possible, en reprenant les propriétés de Figma.
3. Avec JavaScript, **changer la couleur du bouton au click, comme sur le prototype**.
4. Faire un commit, puis un push

## Petit défi supplémentaire

1. Faire que la couleur revienne à la normale après 2 secondes, avec setTimeout
2. Faire qu'à chaque click, un like supplémentaire s'affiche (147, 148, 149, etc)
